---++ Using Google Web Fonts "Google Web Fonts [GWF] lets you browse all the fonts available via the Google Web Fonts API. All fonts in the directory are available for use on your website under an open source license and are served by Google servers." (from the GWF homepage) This is a quite handy tool that will allow us to really open up the design possibilities, font-wise, on the sites we build. Read up more about it, here: http://www.google.com/webfonts A standalone webpage has been built to serve as a companion demo page to this article, here: http://www.funeralnet.com/assets/templates/main/google-web-fonts-demo.html
  1. If and when a GWF is specified in a given website design, the name of that font family will be provided explicitly at the outset of the cutup/development process (or found by reading the font in Photoshop,) and, once the named font to use is known, proceed with these steps:
    • Go to the main GWF homepage and search the page for the target font family name, let's say in this case, 'Amaranth' is specified
    • Once found, click anywhere in the font's row to go to the page dedicated to the font family
    • Click on the 'Use this font' tab
    • In the main body under the tab, check the appropriate box or boxes you will need to satisfy the fonts variation specified in the design/mockup
    • Just below the checkboxes, copy the entire tag
    • Go to the main template file for the website-at-hand and paste the tag copied in the previous step just before the closing </head> tag; if the GWF is being added to a siteZen site, then follow the instructions in the TWiki article entitled, 'Hacking siteZen to Call Supplemental CSS & !JavaScript Files' to write the tag to the header using !JavaScript, post-page-load (or, otherwise use whatever your favorite technique to hack a siteZen site to write the tag to the header that you prefer)
    • Next, add the 'font-family' style attribute to one or more lines in the main stylesheet for the site, attached to the element(s) that should use the font family, like this, for example: h1 { font-family: 'Amaranth', arial, serif; }
    • An even better way to generate the CSS for the font is to click on the 'Launch in font previewer' button on the top left of the page for the font family; this will open a page with a neat interactive tool to style the font - walk through the following hypothetical settings:
      • Adjust the 'Variant' attribute to 'Italic'
      • Adhjust the 'Shadow' attribute to the first option, '2px 2px 2px #aaa'
      • Adjust the 'Spacing' slider bars to increase them a small amount
    • Now, in the text area at the bottom right, copy all of the CSS statements for the 'body' tag that have been generated there and paste them into the main stylesheet
    • Now, preview your page where you are calling your new GWF and bask in the glory!
